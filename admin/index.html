Rows with a manufacturer URL can be auto-filled: the page is fetched through the site's allowlisted proxy, Claude extracts ONLY what the page states (never invents), and you approve each field before it's applied. Needs your Anthropic key in Settings.

Upload photos & map to a product

Pick the product, then drop up to 5 photos. They're resized and compressed right here in the browser. The first photo becomes the product's main image; extra photos go to the product page gallery (tractors). Works with or without a CSV import — photo-only updates are fine. Alt text is generated automatically from the product name on the site.

Need to remove, reorder, or swap photos already on a page — or take a whole product off the site? Use the Product editor — it shows every product's current photos and lets you fix them visually.

Connect / update the accounts
Where do these come from? (one-time, ~30 min, needs the Page admin)
  1. Create a (free) Meta developer app at developers.facebook.com — type “Business”.
  2. In the app, add the Facebook Login for Business / Graph API product and connect the JJ Riggs Page with Andrew's Page-admin login.
  3. Generate a long-lived Page access token with pages_manage_posts (+ instagram_content_publish for Instagram). Page tokens don't expire in practice.
  4. Page ID: on the Facebook Page → About → Page transparency. Instagram account ID: Graph API me/accounts?fields=instagram_business_account — the Instagram account must be a Business/Creator account linked to the Page.
  5. Paste all three here and hit Test. Keys are stored server-side only — never in this browser, never on the public site.
