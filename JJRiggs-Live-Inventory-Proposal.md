# JJ Riggs Equipment — Live Inventory via Google Sheet

Prepared for Andrew Alluis · July 2026 · Aaron Phelps

## What this gets you

One Google Sheet becomes the single place your team updates inventory. Every tractor, mower, and implement on the website lives as a row in that sheet. When something changes on the lot — a model sells out, TYM renames something, a new unit arrives — you change one row, and the website updates to match. No emails back and forth, no waiting on a web edit.

A few things worth calling out:

**Nothing ever gets deleted.** Each product has a Status dropdown — Active, Hidden, or Discontinued. Pulling a model off the site is one click, and it's one click to bring it back. All the "do we keep the T5075?" type decisions become something you control directly.

**Mistake-proofing is built in.** The sheet only accepts valid entries (dropdowns for things like transmission and cab type), flags missing photos in red, and the website update runs a full check before anything publishes — if a row has a problem, it tells us exactly what, and the live site is never touched until it's clean.

**The site stays fast.** The website doesn't depend on Google to load — it reads its own built-in copy of the data, refreshed from your sheet. If Google ever has an outage, your site doesn't notice.

**It starts out already filled in.** We generate the sheet directly from the current site, so day one it matches exactly what's online — including all the model corrections from your June 29 list. Nobody retypes anything.
